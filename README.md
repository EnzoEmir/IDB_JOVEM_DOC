## 💡 Projeto

### 🌐 Plataforma de Eventos e Voluntariado - IDB Jovem

---

## 📖 Documentação

Este repositório contém a documentação do projeto **IDB Jovem**, desenvolvido como parte da disciplina **Técnicas de Programação em plataformas emergentes**.

O sistema tem como objetivo centralizar:

* Eventos da igreja
* Materiais relacionados
* Gestão de voluntários

Além disso, busca melhorar a experiência dos jovens, permitindo acesso rápido às informações mais importantes.

## 🧠 Objetivos

* Desenvolver uma plataforma web funcional para eventos da igreja
* Aplicar metodologias ágeis no desenvolvimento
* Centralizar informações que hoje estão dispersas
* Melhorar o engajamento dos jovens com os eventos
* Criar um sistema escalável com múltiplos administradores

---

## 🚀 Funcionalidades

### 📅 Eventos

* Listagem de eventos (online, presencial ou híbrido)
* Página detalhada com:

  * Data e hora
  * Contagem regressiva
  * Localização (mapa)
  * Palestrantes
  * Bandas
  * Fotos

---

### 🙋 Voluntariado

* Cadastro de voluntários
* Envio de informações:

  * Nome, idade, região, etc.
  * Portfólio (opcional)
* Sistema de aprovação
* Notificação por email para administradores

---

### 👨‍💼 Administração

* Múltiplos administradores (líderes regionais)
* Cadastro e gerenciamento de eventos
* Controle de voluntários

---

### 📚 Materiais

* Disponibilização de conteúdos relacionados aos eventos
* Integração com mídias (fotos, vídeos, etc.)

---

## 👥 Público-Alvo

* Jovens e adolescentes da igreja
* Líderes regionais
* Pastores
* Visitantes interessados em eventos

---

## 📊 MVP (Produto Mínimo Viável)

O MVP do projeto contempla:

* Página inicial com eventos
* Cadastro e listagem de eventos
* Página de detalhes do evento
* Redirecionamento para inscrição externa
* Formulário de voluntariado
* Notificação por email

---

## 🗓️ Cronograma

### 🚀 Sprint 1

* Setup do projeto
* Estrutura base (frontend + backend)
* CRUD de eventos
* Página inicial

---

### 🚀 Sprint 2

* Página detalhada do evento
* Sistema de voluntariado
* Notificação por email
* Integração com links externos

---

### 🚀 Sprint 3

* Sistema de administradores
* Upload de arquivos
* Aba de materiais
* Refinamento de interface

## ⚙️ Pré-requisitos

* Python 3.10+
* Pip instalado

Verificar:

```bash
python --version
```

---

## 📦 Instalação das Dependências

```bash
py -m pip install mkdocs mkdocs-material pymdown-extensions mkdocs-git-revision-date-localized-plugin
```

---

## ▶️ Executando a Documentação

```bash
mkdocs serve
```

Ou no Windows:

```bash
py -m mkdocs serve
```

Acesse:

```
http://127.0.0.1:8000/
```
